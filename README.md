# Collaborative Modding
This is a proof-of-concept for a collaborative modding architecture.

This completely ends mod incompatibility, and informally ends the need for game-specific mod-loaders.

It can mod any apparatus, even if it's not a game.

It's a small-time project, and I am currently playing with this idea, so I decided to put it out there. I'm not taking contributions unless we talk first, and don't expect anything from me (even though it will probably yield)
